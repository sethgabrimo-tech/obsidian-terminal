# Obsidian Terminal

A premium cryptocurrency trading dashboard with brutalist luxury aesthetic.

## Framework

Vite 6.x + React 19.x

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Theme

**Brutalist Luxury meets Cyber-Fintech**

- **Base**: Obsidian black (#08080A) with microscopic silver noise texture
- **Accents**: Electric Jade (#00FFA3) for gains, Molten Ember (#FF3864) for losses, Phosphor Blue (#00D4FF) for whale clusters
- **Typography**: Clash Display (headers), JetBrains Mono (data/numbers)
- **Visual Style**: Sharp corners, layered shadows, ultra-thin hairlines, glass panels with chamfered borders
- **Interaction**: Kinetic snap-in animations, glitch-hover effects on high-volatility assets, analog fader-style sliders

## Features

- **Executive Dashboard**: Asymmetric grid with glass canvas candlestick chart
- **Left Sidebar**: Coin watchlist with sparkline mini-charts
- **Order Book**: Real-time bid/ask spread with depth visualization
- **Shadow Order Entry**: Tactical execution module with slippage protection sliders and gas fee estimator
- **Liquidity Heatmap**: 3D-depth visualization showing whale order clusters
- **Recent Trades**: Real-time trade feed with buy/sell indicators
- **Portfolio Analytics**: Top nav with live portfolio value and 24h change tracking

## Design Philosophy

High-end Swiss watchmaking software meets dark, aggressive, tactical trading interface. No generic SaaS cards, no white backgrounds, no bubbly UI elements. Every interaction is precise, intentional, and premium.
